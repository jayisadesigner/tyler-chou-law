# Dev Content Files

This folder contains markdown files with content for various pages on the site. These files are for reference and content management only - they are not deployed to production.

## Structure

- `content/` - Markdown files for page content
  - `about.md` - About page content
  - `contact.md` - Contact page content
  - `creatorarq.md` - CreatorArq page content
  - `home.md` - Homepage (index.html) content
  - `love-letters.md` - Love Letters blog listing page content
  - `roster.md` - Roster page content
  - `services.md` - Services page content

## Usage

Update the markdown files here with new content, then use them to update the corresponding HTML files in the project root.

## Notes

- These files are version controlled in git
- They are NOT processed by the build system
- They will NOT appear in production builds
- They are safe to edit and commit
