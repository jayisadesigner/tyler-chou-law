# Video Assets

## Intro Animation Videos

Add `tyler-placeholder.mp4` to this directory for the homepage intro animation.

The video should be:
- Format: MP4 (H.264)
- Duration: 5-10 seconds (will loop)
- Aspect ratio: 16:9 or taller
- Optimized for web (compressed, fast-start enabled)

The same video will be used for all three video slots (left, center, right) during development.
Replace with final footage when ready.
